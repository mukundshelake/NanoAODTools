## Selection
This is stage just afterwards our input (i.e. skimmed) datasets.

- It trims the branches to select few
- Filters the data events based on the golden json file.
- It applies following modules
    * LHE Weight Sign
    * Muon ID
    * Muon HLT
    * B-Tagging
